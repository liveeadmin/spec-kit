## Task‑ID Notes Pattern (TODO → Findings)

Use this pattern **before** and **after** each implementation step so `v.memorize` can archive by Task ID.

### [T-XXX] <Task Title>
**TODO (pre‑implementation):**
- [ ] S-XXXX — <first micro‑step>
- [ ] S-XXXX — <second micro‑step>
**Scope:** <files/dirs to touch>  
**Intent:** <outcome you expect>

**Findings (post‑implementation):**
- Summary: <2–4 lines on what changed and why>
- Decisions: <bullets of key choices>
- Evidence: commits <abc123..def456>, PR #, tests green
**Pointer:** ai/memory/archive/T-XXX.md#YYYY-MM-DD
